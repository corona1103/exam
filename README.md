# 在线阅卷诊断系统

一个面向教师的在线试卷批改系统演示，支持按题批改和按学生批改两种模式，包含完整的诊断管理和学生端预览功能。

## 在线体验

**访问地址：** https://corona1103.github.io/exam/

## 功能特性

### 阅卷诊断
- **双模式批改**：支持"按题批改"和"按学生批改"两种工作流
- **丰富的批注工具**：正确✓、错误✗、半对△、圈注、下划线、涂鸦、文字批注
- **批注可拖拽**：添加的批注支持拖动调整位置
- **快捷评分**：一键赋分，提高批改效率
- **学生诊断报告**：自动生成得分统计，支持编写评价建议和老师寄语
- **双诊断类型**：支持"在线模考"和"数据直传"两种模式

### 诊断管理
- **诊断配置**：创建和管理诊断，配置时间、题目、学生范围
- **云控开关**：控制各机构的诊断入口开放状态
- **报名名单**：查看学生报名和答题状态
- **咨询名单**：管理学生的课程咨询申请
- **诊断报告列表**：统一查看和管理已发布的诊断报告

### 学生平板效果
- **iPad 横屏布局**：模拟 1024x680 分辨率的学生端体验
- **完整考试流程**：报名 → 等待 → 开考 → 答题 → 提交 → 查看报告
- **9种卡片状态**：未报名、已报名等待、可开考、考试中、待报告、报告已出等
- **考试界面**：实时倒计时、答题卡面板、交卷确认
- **诊断报告**：得分展示、逐题详情、老师批注回显、课程咨询

## 页面预览

### 诊断列表
选择诊断和批改模式的入口页面，展示诊断类型、进度、状态标签

### 批改界面
- 左侧：题目/学生列表，显示批改进度和状态
- 中间：答卷预览（文字作答或答题卡图片）+ 批注工具栏
- 右侧：评分面板、操作按钮

### 诊断管理
- 统计概览卡片
- 诊断配置列表
- 云控开关弹窗
- 报名/咨询/报告名单弹窗

### 学生平板预览
- 场景切换侧边栏
- iPad 模拟框架
- 诊断列表、考试界面、报告页面
