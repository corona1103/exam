第三题（15分）

已知函数 f(x) = x³ - 3x² + 2，求：

(1) f(x) 的单调区间；

(2) f(x) 在 [-1, 3] 上的最大值和最小值。

学生作答：

解：(1) f'(x) = 3x² - 6x = 3x(x-2)

令 f'(x) = 0，得 x = 0 或 x = 2

当 x < 0 时，f'(x) > 0，单调递增

当 0 < x < 2 时，f'(x) < 0，单调递减

当 x > 2 时，f'(x) > 0，单调递增

(2) f(-1) = -1 - 3 + 2 = -2

f(0) = 2

f(2) = 8 - 12 + 2 = -2

f(3) = 27 - 27 + 2 = 2

所以最大值为 2，最小值为 -2
