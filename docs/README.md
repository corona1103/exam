## 项目概述

在线阅卷系统是一个面向教师的试卷批改工具，支持上传试卷和答题卡后进行在线批改，包括按题批改和按学生批改两种模式，具备评分、批注、评语撰写等功能。

## 快速开始

直接在浏览器中打开 `index.html` 即可运行演示。

```bash
open /Users/tal/Desktop/exam/index.html
```
