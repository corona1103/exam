## 项目概述

在线阅卷诊断系统是一个面向教师的试卷批改工具，包含三大核心模块：

1. **阅卷诊断** - 支持按题批改和按学生批改两种模式，具备评分、批注、诊断报告功能
2. **诊断管理** - 配置诊断、管理报名名单、咨询名单、诊断报告
3. **学生平板效果** - 模拟学生在平板设备上的完整体验流程

## 快速开始

直接在浏览器中打开 `index.html` 即可运行演示。

```bash
open index.html
```

或访问在线演示：https://corona1103.github.io/exam/
