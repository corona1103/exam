# 在线阅卷诊断系统 - 需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
面向学校教师的在线阅卷诊断系统，支持教师对学生试卷进行高效批改、评分和反馈。

### 1.2 目标用户
- 中小学及高中教师
- 教研组长
- 学校管理人员

### 1.3 核心价值
- 提高批改效率：支持按题批改，同题连批
- 灵活批改方式：支持按学生、按题目两种批改模式
- 个性化反馈：支持卷面批注和学生诊断报告
- 报告管理：统一管理和发布学生诊断报告
- 效果预览：模拟学生端查看批改结果

---

## 2. 系统结构

### 2.1 全局导航
系统采用左侧固定导航栏，包含三个主要模块：
| 模块 | 图标 | 说明 |
|------|------|------|
| 阅卷诊断 | 📝 | 核心批改功能，包含诊断列表和批改页面 |
| 诊断管理 | 📊 | 管理诊断配置、报名名单、咨询名单、诊断报告 |
| 学生平板效果 | 📱 | 预览学生端查看成绩的效果 |

## 3. 功能模块

### 3.1 阅卷诊断模块

#### 3.1.1 诊断列表页

**功能描述**
展示所有待诊断和已完成的诊断列表，作为系统入口。

**页面元素**
| 元素 | 说明 |
|------|------|
| 诊断卡片 | 显示诊断名称、科目、班级、日期、试卷份数 |
| 批改进度 | 进度条 + 文字（如 "12/30 已批改"） |
| 状态标签 | 待诊断 / 诊断中 / 已完成 |
| 筛选器 | 按科目筛选、按状态筛选 |
| 操作按钮 | "按学生批改" / "按题批改" |

**交互逻辑**
- 点击"按学生批改"：进入按学生批改模式
- 点击"按题批改"：进入按题批改模式
- 已完成的诊断按钮显示为"按学生查看"/"按题查看"

---

#### 3.1.2 批改页面

**页面布局**
```
+------------------+------------------------+------------------+
|                  |                        |                  |
|    左侧边栏       |      中间答卷区         |    右侧评分面板   |
|   （列表导航）     |    （试卷预览+批注）     |   （评分+记录）   |
|                  |                        |                  |
+------------------+------------------------+------------------+
```

**左侧边栏**
**按题批改模式：**
- 显示题目列表
- 每题显示：题号、满分、批改状态（已批/批改中/待批）
- 点击切换当前批改的题目

**按学生批改模式：**
- 显示学生列表
- 每人显示：姓名、批改进度（如"已批 2/5 题"）、状态
- 学生状态：待诊断 / 诊断中 / 待发布 / 已完成
- 点击切换当前批改的学生

**学生状态说明：**
| 状态 | 说明 |
|------|------|
| 待诊断 | 尚未开始批改 |
| 诊断中 | 正在批改中 |
| 待发布 | 所有题目已批改完成，报告待发布 |
| 已完成 | 报告已发布（只读模式） |

**中间答卷区**
**顶部工具栏：**
- 按题模式：学生导航（上一份/下一份）
- 按学生模式：题目导航（上一题/下一题）
- 缩放控制（放大/缩小）

**答卷预览：**
- 显示题目内容和学生作答
- 支持缩放查看

**底部批注工具栏：**
| 工具 | 功能 |
|------|------|
| 选择 | 拖拽移动已有批注 |
| ✓ | 标记正确 |
| ✗ | 标记错误 |
| △ | 标记半对 |
| 下划线 | 添加下划线标记 |
| 圈注 | 添加圆圈标记 |
| 文字批注 | 添加红色文字批注（支持字号选择） |
| 撤销 | 撤销上一个批注 |
| 清除 | 清除所有批注 |

**批注交互：**
- 点击答卷添加批注
- 选择工具后拖拽可移动批注位置
- 双击批注可删除

**右侧评分面板**
| 模块 | 说明 |
|------|------|
| 评分 | 分数输入框 + 快捷赋分按钮 |
| 批改记录 | 显示当前批改操作历史 |
| 操作按钮 | 保存草稿 / 提交 |

**提交按钮文字逻辑（按学生模式）：**
- 非最后一题：显示"提交并去下一题"
- 最后一题：显示"完成批改并审核报告"

---

#### 3.1.3 学生诊断报告弹窗

**触发条件**
- 按学生批改模式下，完成该学生最后一道题的批改
- 点击"待发布"状态的学生：显示可编辑报告
- 点击"已完成"状态的学生：显示只读报告

**报告内容**

**学生信息区：**
- 学生姓名、班级
- 诊断名称
- 总分 / 满分
- 诊断排名（本次诊断所有学生排名）

**各题得分概览：**
- 柱状进度条展示每题得分比例
- 显示 "得分/满分" 文字

**整体评价建议：**
- 可编辑文本框（已发布状态为只读）
- 教师填写学习建议和评价

**老师寄语：**
- 可编辑文本框（已发布状态为只读）
- 教师填写鼓励语

**操作按钮**
- 待发布状态：保存草稿 / 发布报告，查看下一人
- 已完成状态：仅显示关闭按钮

---

### 3.2 诊断管理模块

#### 功能描述
用于创建、编辑诊断的基础配置，管理诊断入口的开放设置，查看各类名单和报告。

#### 3.2.1 诊断列表页

**页面元素**
| 元素 | 说明 |
|------|------|
| 统计卡片 | 诊断总数、已报名学生、已发报告、课程咨询 |
| 云控开关按钮 | 控制各机构诊断入口开放状态 |
| 诊断列表 | 表格形式展示所有诊断配置 |
| 创建按钮 | 新建诊断配置入口 |
| 分页器 | 支持翻页查看 |

**列表字段**
- 诊断名称
- 诊断类型（在线模考/数据直传）
- 报名开放时间（起止时间范围）
- 考试开放时间（仅在线模考类型显示）
- 学生范围（显示范围和人数）
- 状态（进行中/已结束/待开始）
- 操作按钮（编辑/查看/报名名单/咨询名单/诊断报告）

**编辑规则**
- 已结束的诊断：不可编辑，只能查看
- 进行中的诊断：可编辑，但已报名学生不可移除，只能添加新学生

#### 3.2.2 云控开关

**功能描述**
控制各机构的诊断入口开放状态，关闭后该机构学生将无法访问诊断入口。

**页面元素**
| 元素 | 说明 |
|------|------|
| 机构列表 | 显示机构名称、学生数、当前状态、开关控制 |
| 批量操作 | 全部开放 / 全部关闭 |
| 搜索框 | 搜索机构名称 |
| 统计信息 | 显示开放/关闭机构数量 |
| 操作按钮 | 取消 / 保存并下发 |

#### 3.2.3 诊断配置编辑页

**基本信息**
| 配置项 | 说明 |
|--------|------|
| 诊断名称 | 文本输入，必填 |
| 诊断科目 | 下拉选择，必填 |
| 诊断类型 | 单选：在线模考 / 数据直传 |
| 诊断说明 | 文本输入，选填，最多100字 |

**诊断类型说明**
- **在线模考**：学生在线作答，系统自动收集数据
- **数据直传**：线下考试数据直接导入系统

**时间设置**
| 配置项 | 说明 |
|--------|------|
| 报名开放开始时间 | 诊断入口/报名的开放起始时间 |
| 报名开放结束时间 | 诊断入口/报名的关闭时间 |

**在线模考配置（仅在线模考类型）**
| 配置项 | 说明 |
|--------|------|
| 考试开放开始时间 | 考试开始时间（不能早于报名开始时间） |
| 考试开放结束时间 | 考试结束时间（不能晚于报名结束时间） |
| 考试限制时长 | 学生答题最大时长（分钟） |
| 试卷题目 | 批量添加题目ID，支持逐题赋分 |

**数据直传配置（仅数据直传类型）**
| 配置项 | 说明 |
|--------|------|
| 试卷题目 | 批量添加题目ID（无需赋分） |
| 学生小分表 | 上传 Excel/CSV 格式 |
| 学生答题卡 | 上传图片或 JSON 文件 |

**阅卷范围说明（数据直传）**
只有已报名且上传了答题卡的学生才会纳入阅卷诊断范围，仅有答题卡数据但未报名的学生不会参与阅卷。

**学生范围配置**
三级级联选择：机构 → 班级 → 学生
- 支持搜索精准定位学生
- 支持多选
- 进行中的诊断已报名学生显示"已报名"标签，不可移除

**课程咨询方式**
| 配置项 | 说明 |
|--------|------|
| 咨询说明 | 富文本编辑器，支持加粗、斜体、链接、图片 |
| 需要提交手机号预约 | 复选框，勾选后学生需填写手机号才能报名 |

**操作按钮**
- 取消：放弃编辑，返回列表
- 保存草稿：暂存配置内容
- 发布诊断：发布配置，开放入口

#### 3.2.4 报名名单弹窗

**列表字段**
- 序号、机构、学生姓名、学号、班级、报名时间、答题状态

**答题状态**
| 状态 | 说明 |
|------|------|
| 未提交 | 已报名，正在答题 |
| 阅卷中 | 已提交答题/已上传答题卡，等待阅卷 |
| 报告已发布 | 已完成阅卷并发布报告 |

#### 3.2.5 咨询名单弹窗

**列表字段**
- 序号、学生姓名、学号、班级、诊断得分、手机号、预约时间、提交时间

**预约时间选项**
- 工作日白天/晚上
- 周六白天/晚上
- 周日白天/晚上

#### 3.2.6 诊断报告列表弹窗

**列表字段**
- 序号、学生姓名、学号、班级、诊断得分、诊断排名、完成时间、操作

**说明**
- 诊断排名为本次诊断所有学生的排名
- 点击"查看报告"可查看学生诊断报告详情

---

### 3.3 学生平板效果模块

#### 功能描述
模拟学生在平板设备上查看批改结果和诊断报告的效果，方便教师预览学生端体验。

#### 页面元素
| 元素 | 说明 |
|------|------|
| 平板框架 | 模拟平板设备外观 |
| 学生端界面 | 显示成绩、各题得分、老师评价、寄语 |
| 说明文字 | 介绍页面用途 |

#### 学生端展示内容
- 顶部：应用标题 + 学生姓名
- 成绩卡片：诊断名称、大号分数、诊断排名（本次诊断所有学生）
- 各题得分：进度条形式展示
- 老师评价：整体评价建议
- 老师寄语：带签名的寄语卡片
- 操作按钮：查看答卷详情

## 4. 业务流程

### 4.1 按题批改流程
```
选择诊断 → 点击"按题批改" → 选择题目 → 批改学生A该题 → 提交
→ 自动切换到学生B该题 → ... → 该题所有学生批完 → 选择下一题
```

### 4.2 按学生批改流程
```
选择诊断 → 点击"按学生批改" → 选择学生 → 批改第1题 → 提交并去下一题
→ 批改第2题 → ... → 完成最后一题 → 弹出诊断报告 → 编辑评价和寄语
→ 发布报告 → 自动切换到下一个学生
```

### 4.3 特殊场景
- 点击"待发布"状态学生：直接弹出可编辑报告
- 点击"已完成"状态学生：弹出只读报告
- 发布报告后下一个学生也是待发布状态：自动弹出该学生报告

## 5. 数据结构

### 5.1 诊断信息
```javascript
{
  id: 1,
  name: '2024年期中考试 - 数学',
  subject: 'math',
  description: '诊断说明（选填）',
  type: 'online', // online | direct
  enrollStartTime: '2024-04-15 00:00',
  enrollEndTime: '2024-04-20 23:59',
  examStartTime: '2024-04-16 08:00', // 仅在线模考
  examEndTime: '2024-04-19 18:00',   // 仅在线模考
  examDuration: 120, // 分钟
  status: 'active' // pending | active | ended
}
```

### 5.2 题目信息
```javascript
{
  id: 1,
  questionId: 'Q001',
  type: 'choice', // choice | multi | fill | essay | judge
  fullScore: 10,
  knowledgePoint: '函数求导',
  content: '题目内容...',
  standardAnswer: '标准答案...'
}
```

### 5.3 学生答卷
```javascript
{
  studentId: 1,
  studentName: '李明',
  questionId: 1,
  answer: '学生作答内容...',
  score: 8,
  annotations: [...], // 批注数组
  status: 'graded' // pending | grading | graded
}
```

### 5.4 诊断报告
```javascript
{
  studentId: 1,
  examId: 1,
  totalScore: 62,
  rank: 8, // 诊断排名
  totalStudents: 285,
  questionScores: [10, 8, 12, 13, 19],
  overallComment: '整体评价建议...',
  teacherMessage: '老师寄语...',
  status: 'published' // draft | pending | published
}
```

## 6. 非功能需求

### 6.1 性能要求
- 页面加载时间 < 2秒
- 批注操作响应时间 < 100ms
- 支持单次加载 50+ 份试卷

### 6.2 兼容性
- 支持 Chrome、Firefox、Safari、Edge 最新版本
- 支持 1280px 及以上屏幕宽度

### 6.3 可用性
- 操作反馈及时（Toast 提示）
- 支持键盘快捷操作（ESC 关闭弹窗）
- 批注支持拖拽调整位置
- 时间配置支持校验（考试时间需在报名时间范围内）
